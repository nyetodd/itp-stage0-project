Web basics

Some background on how the web works

The world wide web is made up of HTML documents. HTML stands for Hyper Text Markup Language. These are documents are linked together though the use of hyperlinks.

Major pieces of the web

Browser: The browser is the tool that a person uses to makes requests via the internet to servers. Web content is viewed on a browser (such as Chrome, Firefox etc)
The internet: A massive global computer network that allows data and documents to be retrieved and viewed by using standard protocols such as HTTP
Servers: Servers store files and content. When they receive requests they identify what is being requested and pass this back.

An example: If you enter the url for the video streaming site YouTube into your browser, this request is passed using the HTTP protocol to servers which then return the HTML page for that website which is rendered by your browser for you to see.

HTML and Markup

HTML is made up of the following elements:

  • Text content: what is written on the page
  • Markup: tags that decide how this content is displayed
  • References to other documents
  • Links to other pages

A bit more on markup
Markup is made up of tags. An opening tag will be placed at the start of the text or item that we want to change the look of, and the closing tag at the end. The only difference is that the closing tag has a forward slash '/' The tags and what is within them can be referred to as the 'element'. If you forget to add a closing tag when applying a style to text (eg. making it bold with the 'b' tag) then the style will be applied to all text on the page.
